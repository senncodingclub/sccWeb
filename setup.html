Senn Coding Club - Intermediate Project Setup

Intermediate Coding Club members have their own web space where their work will get pushed. To get your project space setup, follow these steps:

  1. Request Web space from an administrator (JC or Rakesh).
  2. Set up an account on GitHub.
  3. In GitHub, create a repository called "".
  4. On your local computer, create a folder called "sccgmaes" in your download folder.
  5. Download the following files:
    • game.html
    • gitupdate.php_
    • game.js
  6. Change the "gitupdate.php_" to "gitupdate.php"
  • Using the Git command line, issue the following commands from within your download folder:
    • git clone https://github.com/username/repository . (change username to your GitHub username).
    • git config --global user.name "YOUR NAME"
    • git add game.html
    • git add gitupdate.php
    • git add game.js
    • git commit -m"Initial setup"
    • git push
  • Inform an administrator that your repository is setup. The admin will get your repo into the project space.
  • In GitHub, go to Settings, Webhooks. Add the following Web hook URL: https://senncodingclub.org/projects/username/gitupdate.php (replace username with your Senn Coding Club user name.
  • Change the content-type to application/x-www-form-urlencoded.
  • Your project space will be available at https://senncodinglcub.org/projects/username/
